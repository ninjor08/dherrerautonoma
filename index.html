Definition

The document object in JavaScript is the browser's main object that represents the entire web page. It allows you to access and modify HTML elements using properties and methods (Escarcena, 2020).

The DOM structure is hierarchical:
*Elements (div, h1, p) are nodes.
*The content within them is a text node.
*Each node has relationships: parent, child, and siblings.


Characteristics

Full access to the web page

You can read, modify, or add content using JavaScript.

Element selection methods

getElementById(), getElementsByTagName(), getElementsByName().

Tree-like structure

The DOM organizes elements as nodes in a parent-child hierarchy.

Content modification

Use innerHTML, setAttribute(), or .style to change elements.


Objectives

The DOM enables developers to create more dynamic, interactive, and accessible web applications. Its objective is to give full control over how web documents are structured and manipulated.
